# Berlin Space: Parks vs. Parking

A minimalist map visualization comparing spatial equity in Berlin by overlaying public green spaces (parks) against land dedicated to car infrastructure (parking). 

## Run Locally

This project uses native HTML/JS/CSS without a bundler, but it uses relative paths for map styles. 

To view the project locally without CORS errors:
1. Clone the repository.
2. Serve the root directory using a local web server. For example: `python3 -m http.server 8000` or use VS Code's Live Server extension.
3. Open `http://localhost:8000` (or your chosen URL) in your browser.
